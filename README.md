### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/jkrmarmol/portfolio-v1.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Run App
    ```sh
    npm start
    ```
